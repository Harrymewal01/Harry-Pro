== Description ==

RMA - Return Refund Exchange for WooCommerce Pro is a fully functional WordPress plugin that provides return, refund, and exchange features for WooCommerce stores. 

**Key Features:**
- Customers can request refunds, exchanges, or returns from their account.
- Admin can approve or reject requests.
- Email & WhatsApp notifications for updates.
- Supports automatic return label generation.
- Customers can upload photos for verification.

== Installation ==

1. Upload the `rma-return-refund-exchange-for-woocommerce-pro.zip` file to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings under WooCommerce > RMA Settings.
